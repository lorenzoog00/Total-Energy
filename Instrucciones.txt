Instrucciones para correr el programa
1. Instalar python
2. Meterse al link
3. Descargar carpeta que se llama ejecutable
4. meterse al directorio de ejecutable y poner en la terminal:
	Windows: pip install -r requirements.txt
	Mac/Linux: pip3 install -r requirements.txt
5. Despues que se instalen todos los paquetes, poner en la terminal: python app.py
6. La aplicacion empezara a correr y cuando salga el IP de la computadora darle click
